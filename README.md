# Weatherly01 – Weather Dashboard

A real‑time weather app with dynamic backgrounds, 5‑day forecast, recent search history, and error handling. Uses the OpenWeatherMap API and async/await.

**Live Site:** [https://weatherly01.netlify.app](https://weatherly01.netlify.app)

## Features
- Search weather by city name
- Current conditions: temperature, description, humidity, wind speed, feels‑like
- 5‑day forecast with daily high/low temperatures
- Dynamic background gradients that change based on weather (sunny, cloudy, rainy, etc.)
- Night‑time detection for appropriate dark gradients
- Recent searches saved in local storage (up to 5)
- Clickable recent search chips
- Weather emoji in the header that updates with conditions
- Error handling for invalid city or network issues
- Loading state while fetching data
- Modern glassmorphism UI, fully responsive

## Built With
- HTML5
- CSS3 (custom properties, backdrop‑filter, transitions)
- JavaScript (ES6+, async/await, fetch API)
- OpenWeatherMap API

## Setup
1. Clone the repo.
2. Get a free API key from [OpenWeatherMap](https://openweathermap.org/api).
3. Replace `YOUR_API_KEY` in `index.html` with your actual key.
4. Open `index.html` in your browser.

*Note: The live site already uses a valid API key.*
